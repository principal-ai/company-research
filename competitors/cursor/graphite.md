# Graphite (Cursor / Anysphere)

An AI code-review product centered on **stacked pull requests**, acquired by
[Anysphere](index.md) in December 2025 and being folded into the Cursor stack.

## Overview

- **Vendor:** Anysphere, Inc. (acquired Graphite, Dec 2025).
- **Category:** AI code review / developer workflow.
- **Traction claim:** the company said Graphite served **100K+ users across 500+
  companies** as of August 2025.

## Features

- **Stacked pull requests** — work on multiple dependent changes simultaneously,
  broken into smaller reviewable increments.
- **AI code review** — automated review assistance over pull requests.

## Competitive surface

- **Code review / understanding** — review is squarely inside Principal's domain
  (understanding and collaborating on code), and is the bottleneck Cursor cites as
  its rationale for the acquisition.

How these compare to Principal is a trail concern, not recorded here.
