# Cursor (editor)

The core Cursor product: an AI-native code editor built as a fork of VS Code,
made by [Anysphere](index.md).

## Overview

- **Vendor:** Anysphere, Inc.
- **Product:** Cursor editor (VS Code fork).
- **Category:** AI-native IDE.

## Features

- **Agent workflows** — chat, agent mode, and multi-file edits over a repo.
- **Inline assistance** — tab completion and context-aware suggestions.
- **Model choice / MAX mode** — heavier models and longer context on demand.

## Pricing

Cursor moved from request caps to **usage-based billing** in 2025; the change and
the surrounding backlash are recorded as a market signal — see
[`market/cursor-pricing-2025.md`](../../market/cursor-pricing-2025.md). Not
restated here (single home for the fact).

## Competitive surface

- **Code understanding / context** — the editor builds and uses repo context to
  drive edits, which is adjacent to Principal's understanding/context domain.
