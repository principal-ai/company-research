# Origin (Cursor / Anysphere)

**Origin** is [Anysphere](index.md)'s announced **agent-native, Git-compatible
code-hosting platform** — a forge built on the assumption that many AI agents
clone, branch, commit, rebase, review, and fix in parallel. It is positioned as a
direct GitHub challenger. **Status: announced / in development, not yet GA.**

## Overview

- **Vendor:** Anysphere, Inc.
- **Category:** Git hosting platform / forge (agent-native).
- **Status:** announced; availability targeted for **fall 2026** (not GA).
- **Leadership:** built by ex-Graphite engineers (incl. Tomas Reimers).

## Features (announced)

- **Agent-native Git forge** — Git-compatible hosting designed for many parallel
  AI agents rather than human-paced workflows.
- **Replaces GitHub core functions** — repository hosting, security review, and
  automated testing are the stated targets.
- **Completes the stack** — host + merge layer beneath the Cursor editor and
  [Graphite](graphite.md) review.

## Competitive surface

- **Code hosting + review + agent collaboration** — Origin would put Cursor in
  control of the repository as the system of record, plus review and collaboration
  — overlapping Principal's understanding/context/collaboration domain.

How these compare to Principal is a trail concern, not recorded here.
