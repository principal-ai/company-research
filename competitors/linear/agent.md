# Linear Agent

An AI agent built into [Linear](index.md), announced **March 24, 2026** as "the
next major step in Linear's evolution." It is grounded in the workspace and can
"help you synthesize context, make recommendations, and take action" across
roadmaps, issues, and code.

## Overview

- **Vendor:** Linear.
- **Product:** Linear Agent — **public beta** (launched Mar 2026, for all teams).
- **Category:** in-product AI agent / workspace assistant.

## Features

- **Workspace-grounded actions** — synthesizes context to convert Slack
  discussions into issues, generate project update summaries, surface recurring
  themes in backlogs, and flag at-risk work.
- **Access points** — desktop app (`Cmd/Ctrl + J`), mobile, Slack (`@Linear`),
  Microsoft Teams, and inline in Linear comments; can process uploaded content
  such as videos.
- **Skills** — users can save effective workflows as reusable **Skills**.
- **Automations** — trigger agent workflows on events (e.g. when issues enter
  triage); available on Business and Enterprise plans.
- **Code Intelligence** — announced as forthcoming: code-aware tasks like
  diagnosing app functionality, aimed at non-technical teammates; coming to
  Business and Enterprise.

## Pricing

Chat features are included in base pricing; **Automations** and **Code
Intelligence** are announced as eventually moving to **usage-based pricing beyond
a certain threshold**. Models/providers are not specified in the announcement.

## Competitive surface

- **Code-aware tasks (Code Intelligence)** — diagnosing/answering about app
  behavior from inside the work-tracking surface overlaps Principal's
  understanding domain.
- **Ticket → agent → action loop** — assigning work to an agent from an issue is
  the front half of the production-signal → code → review loop Principal also
  addresses.

How these compare to Principal is a trail concern, not recorded here.
