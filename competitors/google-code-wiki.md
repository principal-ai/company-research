# Google Code Wiki

**Code Wiki** is a Google product, powered by the **Gemini** model family, that
"ingests public repositories and generates, hosts, and maintains comprehensive and
interactive documentation for each one." It is described as Google's first product
built on a new code-understanding system. It is **separate from Gemini Code
Assist** (Google's IDE coding assistant).

## Overview

- **Vendor:** Google.
- **Product:** Code Wiki — **public preview**, launched **Nov 13, 2025**, at
  [codewiki.google](https://codewiki.google).
- **Category:** AI code documentation / codebase comprehension.

## Features

- **Living documentation** — "maintains a continuously updated, structured wiki for
  code repositories"; docs regenerate as code changes so they stay current.
- **Code-linked navigation** — jump from high-level concept explanations directly
  to the exact files, classes, and functions they reference; sections and chat
  answers are hyperlinked to code.
- **Gemini chat Q&A** — "ask the Gemini-powered chat agent that uses the
  always-up-to-date wiki as context to answer highly specific questions about
  *your* repository."
- **Auto-diagrams** — "automatically generates always-current architecture, class,
  and sequence diagrams" that match the current state of the code.
- **Repos** — public GitHub repositories at launch; a **Gemini CLI extension for
  Code Wiki** is "coming soon" to "run the same system locally and securely on
  internal repositories" (waitlist).

## Pricing

No pricing or supported-language list was stated in the launch announcement; the
web product is open for public repositories during preview.

## Competitive surface

The Code Wiki features that fall inside Principal's domain — understanding,
documentation, context:

- **Auto-generated, living, code-linked docs + diagrams** — turning a repo into
  navigable, always-current documentation overlaps Principal's File City /
  understanding territory.
- **Codebase Q&A grounded in the repo** — answering questions about a specific
  codebase touches Principal's understanding surface.

How these compare to Principal is a trail concern, not recorded here.
