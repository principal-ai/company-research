# company-research

This is an **OKF** (Open Knowledge Format) bundle — markdown + YAML frontmatter,
one concept per file, organized so each fact stays *pointable* and trails can
thread those facts into grounded, auditable answers.

👉 **Start with [`index.md`](index.md)** for what this bundle is, how it's
organized, and the trails it's built to carry. The hard format rules live in
[`OKF.md`](OKF.md); authoring conventions live in
[`.claude/skills/knowledge-base/SKILL.md`](.claude/skills/knowledge-base/SKILL.md).

## About OKF

OKF is Google Cloud's vendor-neutral standard (v0.1, Apache 2.0) for giving AI
agents curated context as a directory of markdown + YAML-frontmatter files — one
concept per file, readable by humans, parseable by agents, diffable in git.

- Announcement: <https://cloud.google.com/blog/products/data-analytics/how-the-open-knowledge-format-can-improve-data-sharing/>
- Spec & reference implementations: <https://github.com/GoogleCloudPlatform/knowledge-catalog/tree/main/okf>
