# Filed patents

Principal AI has filed **10 distinct patent applications** covering the technical primitives and systems underlying Code Trails, File City, and the broader agentic software development platform. All three founders (Julie Allen, Fernando Ramirez, Michael Flannery) are listed as inventors on all patents. Filed between October 2025 and February 2026.

## Context Scaffolding System (Patent #1)

**File:** `Context Scaffolding System for Facilating Agentic Software Deveopment.pdf`
**Provisional:** 63/907,051 (filed Oct 28, 2025)

System for providing pre-computed, expert-curated context to agentic software development tools. Addresses the problem of agents re-deriving context from raw source on every invocation by maintaining structured scaffolding that agents can consume directly.

## Continuous Architectural Supervision (Patent #2)

**File:** `Continuous Arch Supervision.pdf`

Methods for continuous monitoring and supervision of architectural decisions made by AI agents during software development. Includes normalized graph representation, temporal interpolation, and AI supervision interfaces across platforms. Enables real-time oversight of agent-driven architectural changes.

## Runtime Log-Driven Architectural Animation and Behavioral Validation (Patent #3)

**File:** `Animation_Behavioral_Validation_USPTO_FINAL-SPEC.pdf`

System for animating software architecture based on runtime logs and validating behavioral contracts. Binds intent (architectural design) to runtime telemetry for validation that systems behave as specified. The runtime grounding layer that extends trails from source into production.

## Bidirectional Contracts (Patent #4)

**File:** `Patent_7_Bidirectional_Agent_USPTO_FINAL-SPEC.pdf`

Framework for bidirectional contracts between human intent and agent execution. Establishes verifiable agreements where agents commit to behavioral constraints and humans provide architectural guidance. The "behavioral contracts bridge the gap" between generation speed and comprehension.

## Story Mining & Training-Ready Telemetry (Patent #5)

**File:** `Patent_9_CodebaseMining_USPTO_FINAL-SPEC.pdf`

Methods for mining codebases to extract training-ready telemetry and narrative structures. Produces machine-readable reasoning chains over code as a byproduct of already-useful work — the "expert reasoning as exhaust" mechanism underlying the data moat (act two of the thesis).

## Curated Gallery Presentation (Patent #6)

**File:** `Repository_Gallery_System_FINAL.pdf`

Curated gallery presentation system for pre-computed architectural and multi-dimensional quality analyses of software repositories. Enables instant exploration of repository characteristics without requiring analysis at query time. Maps to the Repository Gallery System product capability.

## Intent-First Telemetry Validation (Patent #7)

**File:** `Patent_Workflow_Validation_USPTO_FINAL.docx`

Intent-first telemetry validation with production-to-development feedback loops. Validates that production workflows conform to curated contracts, with anomalies feeding back to development. Contract validation enforces telemetry conformance — the runtime grounding enforcement mechanism.

## AI-Assisted Architecture Understanding (Patent #8)

**File:** `Patent_AIRepoAgent_USPTO.docx`

System for AI-assisted software architecture understanding using repository-grounded reasoning and synchronized documentation generation. Binds architectural understanding to actual code structure, ensuring documentation stays grounded as code evolves.

## Composable Narrative Unit Architecture (Patent #9)

**File:** `Narrative_Intelligence_Patent_v6 copy.docx`

Systems and methods for composable narrative unit architecture and algebraic operations on narrative graphs. Domain-agnostic architecture for representing organizational assertions as machine-readable, typed narrative units with defined dependency relationships, automated cross-layer propagation, and formal algebraic operations. The primitive that enables narrative coherence across operational domains.

## Universal Code Quality Orchestration (Patent #10)

**File:** `Patent_CodeQuality_USPTO copy.docx`

Universal code quality orchestration using a lens abstraction framework with cross-platform execution. Lens abstraction layer defines a common interface (isApplicable, discoverConfig, execute, parse, extractMetrics) that orchestrates heterogeneous quality tools, executes across diverse runtime environments, and presents unified multi-dimensional quality visualizations. Related application claiming benefit of provisional 63/907,051.

**Supervision & orchestration:**

- Code quality orchestration (lens abstraction, cross-platform execution)

**Visualization & presentation:**
- Repository gallery system (pre-computed multi-dimensional analyses)

**Data asset:**
- Story mining / codebase mining (training-ready telemetry as exhaust)

## Public disclosure status

Provisional patent applications are **not publicly searchable** in USPTO databases — they remain confidential unless/until a non-provisional application is filed and published (typically 18 months after earliest filing date). Based on filing timeline (Oct 2025 - Feb 2026), non-provisional publications would appear between April 2027 and August 2027.

The only confirmed provisional number from the documents: **63/907,051** (Context Scaffolding, filed Oct 28, 2025), which is referenced by the Code Quality patent as a related application.
