# File City tours — public demonstration channel

A **File City tour** is a short, public walkthrough of a real codebase rendered
in **File City** (Principal's visualization layer — files as buildings, 2D map
primary). Principal posts these tours on a dedicated page as the channel for
showing the visualization in action on recognizable codebases.

## What a tour is

A guided pass over a repository in File City: the spatial layout, trail markers
landing on their buildings, source slices opening, and repo activity surfaced on
the map. Each tour is a self-contained demonstration of the visualization on one
codebase.

## The channel

The tours are published on Principal's File City tour page on X/Twitter,
[@FileCityAI](https://x.com/FileCityAI). It is
part of the company's **public presence** — a running, public demonstration reel
rather than internal product documentation. The visualization layer it shows is
described in [`../product/file-city.md`](../product/file-city.md).

## Why it's here, not as an argument

This doc records the channel as a fact. The claim that the tours **prove** the
value of the visualization is a value argument — that lives in a **trail** across
this doc and [`../product/file-city.md`](../product/file-city.md), not in prose
here.
