# GitHub Copilot move to usage-based billing (June 2026)

One aspect per heading.

## The change
On 2026-06-01, all GitHub Copilot plans moved to usage-based billing. Each plan
includes a monthly allotment of "GitHub AI Credits," with paid plans able to buy
more; usage is calculated on token consumption (input, output, and cached tokens)
at each model's listed API rates. Headline prices were held: Pro $10/mo, Pro+
$39/mo, Business $19/user/mo, Enterprise $39/user/mo.

## The backlash
Developer response on GitHub's forums, Reddit, and X was largely negative,
centered on how fast credits deplete and how unpredictable per-request costs are.
One developer reported using 8% of their monthly credits in two hours of normal
work. The dominant complaint was unpredictability rather than the existence of
usage pricing.
