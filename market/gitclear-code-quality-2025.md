# GitClear 2025 AI Copilot Code Quality report

GitClear analysis of **211 million changed lines** of code spanning January 2020
to December 2024, drawn from repositories including Google, Microsoft, Meta, and
enterprise C-corps. One finding per heading.

## Code cloning up ~4x
Code cloning rose roughly fourfold in 2024 versus prior years. "Copy/paste" code
rose from 8.3% to 12.3% of changed lines between 2021 and 2024.

## Refactoring in decline
The share of changed lines associated with refactoring fell from 25% in 2021 to
under 10% in 2024.

## Copy/paste overtook reuse
For the first time in GitClear's measurement history, "copy/paste" code exceeded
"moved" code (a proxy for reuse) — a continued decline in code reuse.
